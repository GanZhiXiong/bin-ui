## 介绍

bin-ui 是一款基于 Vue.js 2.0 的前端 UI 组件库，主要用于快速开发 PC 网站中后台产品

### 特性
- 基于 Vue 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件
- 提供了一些常用的字体图标
    - [Icon 图标](/#/icon)
- 集成了常用的工具库如
    - normalize.css
  -   [flex.css](https://github.com/lzxb/flex.css)
- 封装了常用的指令
    - [水波纹指令](/#/directive)
    - [字数限制指令](/#/directive)

### 浏览器支持
- 现代浏览器和 IE9 及以上
